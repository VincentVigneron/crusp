Traits

Array

This trait describes an array of variables. There is two types of array: array of variables and array of references to variables. Both types are manipulated with the same trait. When writting constraints over an array of variables, you should use the Array trait instead of the specific types ArrayOfVars or ArrayOfRefs.

Variable

Trait for types that represent decision variables. A decision variable is variable along side with its domain of allowed values. A variable has to be cloneable because the (tree based) searching process is based on cloning.
